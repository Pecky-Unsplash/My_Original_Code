# Code availability

This repository contains the original analysis code for the PBMC-reference normalization framework used to rank placenta-associated chromatin accessibility above an immune-cell background, and its statistical validation.

Code is provided for:

(1) width-weighted quantification of bulk ATAC-seq signal across gene-centered and promoter-centered windows; 

(2) extraction of single-cell PBMC reference accessibility per immune cell group; 

(3) the PBMC-normalization and gene-ranking framework (the central methodological contribution of this study); and 

(4) permutation-based statistical validation of the top-ranked subset.

Standard preprocessing of the public single-cell PBMC multiome reference (quality control, dimensionality reduction, and reference-based cell-type label transfer) followed published Signac and Seurat workflows (Stuart et al., 2021; Hao et al., 2021) and is not included here, as it does not constitute original code. 

Figure 4 was generated by visualizing bulk ATAC-seq bigWig tracks alongside public annotation tracks (ENCODE cCREs, FANTOM5 CAGE, GeneHancer) using the UCSC Genome Browser and is not produced by R code.

Input data are not included in this repository. The scripts require normalized bulk maternal PBMC ATAC-seq bigWig files and a processed single-cell PBMC reference object as described in the manuscript. Access to study-generated sequencing data is subject to collaborative data-sharing agreements associated with the ongoing parent study.
